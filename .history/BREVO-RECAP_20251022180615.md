## 🚀 Actions à faire MAINTENANT (10 minutes)

### ✅ Étape 1 : Créer compte Brevo (2 min)
1. Allez sur https://www.brevo.com/fr/
2. S'inscrire gratuitement avec `lesptitsloupiotscharentais@gmail.com`
3. Valider l'email reçu

### ✅ Étape 2 : Obtenir clé SMTP (3 min)
1. Connexion Brevo
2. Paramètres → SMTP & API → SMTP
3. Cliquer "Créer une nouvelle clé SMTP"
4. Nom : "Site Web Les Ptits Loupiots"
5. **COPIER la clé** (format : `xkeysib-abc123...`)

### ✅ Étape 3 : Configuration locale (1 min)
Ouvrir `.env.local` et remplacer :
```bash
BREVO_SMTP_KEY=votre_cle_smtp_brevo_ici
```
Par votre vraie clé :
```bash
BREVO_SMTP_KEY=xkeysib-abc123def456...
```

### ✅ Étape 4 : Configuration Vercel (2 min)
1. https://vercel.com/dashboard → Votre projet
2. Settings → Environment Variables
3. **Supprimer** :
   - ❌ `GMAIL_USER`
   - ❌ `GMAIL_APP_PASSWORD`
4. **Ajouter** :
   - `BREVO_SMTP_USER` = `lesptitsloupiotscharentais@gmail.com`
   - `BREVO_SMTP_KEY` = (votre clé)
   - ✅ Cocher : Production + Preview + Development

### ✅ Étape 5 : Déployer (2 min)
```bash
git add .
git commit -m "Migration vers Brevo SMTP"
git push
```
Attendre 2-3 min (déploiement Vercel)

### ✅ Étape 6 : Tester
1. https://lesptitsloupiotscharentais.fr/contact
2. Envoyer un message test
3. Vérifier Gmail → Email reçu ! ✅
